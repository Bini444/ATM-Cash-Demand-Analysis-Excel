ATM Cash Demand Analysis - Excel Dashboard

Project Overview
Analysis of 6 years of ATM transaction data (2011-2017) to identify cash demand patterns and optimize ATM cash replenishment strategies.

Dataset: 11,589 transaction records from Kaggle

Key Findings
Saturday demand spikes 13% above average
One ATM handles 31% of network volume
Month-start creates 25% surge in residential areas
Inter-bank withdrawals nearly match own-bank transactions

Current Version: Excel Dashboard

Interactive dashboard built in Excel featuring:
20+ calculated fields for weekday/payday/location analysis
XLOOKUP and INDEX-MATCH functions for data merging
8 pivot tables with interactive slicers
Visual insights on when and where cash demand peaks
